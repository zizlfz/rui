# DateField

A date input field that allows users to enter and edit dates using localized date segments (month, day, year).

## Installation

```bash
rui add datefield
```

## Usage

DateField requires child components to render the input segments. Use `DateInput` with a render function that returns `DateSegment` for each segment.

<DocDemo>
  <DateField label="Birth date">
    <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
  </DateField>
</DocDemo>

```tsx
import { DateField, DateInput, DateSegment } from '@/components/ui/DateField';

<DateField label="Birth date">
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Value

A `DateField` has no value by default. An initial, uncontrolled value can be provided using the `defaultValue` prop. Alternatively, a controlled value can be provided using the `value` prop.

Date values are provided using objects from the [@internationalized/date](https://react-spectrum.adobe.com/internationalized/date/) package.

<DocDemo>
  <DateField label="Birth date">
    <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
  </DateField>
</DocDemo>

```tsx
import { parseDate } from '@internationalized/date';
import { DateField, DateInput, DateSegment } from '@/components/ui/DateField';

// Uncontrolled with defaultValue
<DateField label="Event date" defaultValue={parseDate('2024-12-25')}>
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>

// Controlled with value
function ControlledDateField() {
  const [value, setValue] = useState(parseDate('2024-01-15'));

  return (
    <DateField label="Event date" value={value} onChange={setValue}>
      <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
    </DateField>
  );
}
```

## Label and Description

Use the `label` prop to provide an accessible label. The `description` prop adds helper text below the field.

<DocDemo>
  <DateField label="Meeting date" description="Select a date for your meeting.">
    <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
  </DateField>
</DocDemo>

```tsx
<DateField label="Meeting date" description="Select a date for your meeting.">
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Min and Max Dates

Use the `minValue` and `maxValue` props to constrain the selectable date range.

```tsx
import { today, getLocalTimeZone } from '@internationalized/date';

<DateField label="Appointment date" minValue={today(getLocalTimeZone())}>
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Validation

Use the `isRequired` prop to require a value. The `errorMessage` prop provides a custom message displayed when validation fails.

```tsx
<DateField label="Event date" isRequired errorMessage="A date is required.">
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Disabled

Use the `isDisabled` prop to disable the date field. It cannot be interacted with and is excluded from the tab order.

<DocDemo>
  <DateField label="Date" isDisabled>
    <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
  </DateField>
</DocDemo>

```tsx
<DateField label="Date" isDisabled>
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Read Only

Use the `isReadOnly` prop to allow viewing the selected date without allowing changes.

```tsx
<DateField label="Created" value={parseDate('2024-09-05')} isReadOnly>
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Granularity

The `granularity` prop allows specifying the smallest unit displayed. Use `"hour"`, `"minute"`, or `"second"` to include time segments alongside the date.

```tsx
<DateField label="Start time" granularity="minute">
  <DateInput>{(segment) => <DateSegment segment={segment} />}</DateInput>
</DateField>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Arrow Up` / `Arrow Down` | Increments / decrements the focused date segment. |
| `Arrow Left` / `Arrow Right` | Moves focus to the previous / next date segment. |
| `Page Up` / `Page Down` | Increments / decrements by a larger unit (month or year). |
