# Checkbox

A checkbox allows a user to select a single item from a list of options, or to mark an item as true or false. It supports an indeterminate state for representing partial selections.

## Installation

```bash
rui add checkbox
```

## Usage

<DocDemo>
  <Checkbox>Subscribe to newsletter</Checkbox>
</DocDemo>

```tsx
import { Checkbox } from '@/components/ui/Checkbox';

<Checkbox>Subscribe to newsletter</Checkbox>
```

## Value

Use the `isSelected` and `onChange` props to control the checked state.

<DocDemo props={{ isSelected: false }} propsForm={{ isSelected: Boolean }}>
  {(props) => <Checkbox {...props}>Accept terms</Checkbox>}
</DocDemo>

```tsx
function ControlledCheckbox() {
  const [isSelected, setIsSelected] = useState(false);

  return (
    <Checkbox isSelected={isSelected} onChange={setIsSelected}>
      Accept terms
    </Checkbox>
  );
}
```

An initial, uncontrolled value can be provided using the `defaultSelected` prop.

```tsx
<Checkbox defaultSelected>Accept terms</Checkbox>
```

## Indeterminate

Use the `isIndeterminate` prop to display a partially selected state, commonly used in tree views or when a parent checkbox represents a mixed selection of child checkboxes.

<DocDemo>
  <Checkbox isIndeterminate>Select all</Checkbox>
</DocDemo>

```tsx
<Checkbox isIndeterminate>Select all</Checkbox>
```

## Disabled

Use the `isDisabled` prop to disable the checkbox. It cannot be interacted with and is excluded from the tab order.

<DocDemo>
  <Checkbox isDisabled>Disabled option</Checkbox>
</DocDemo>

```tsx
<Checkbox isDisabled>Disabled option</Checkbox>
```

## Read Only

Use the `isReadOnly` prop to allow viewing the checked state without allowing changes.

```tsx
<Checkbox isSelected isReadOnly>Read-only option</Checkbox>
```

## Invalid

Use the `isInvalid` prop to indicate a validation error.

<DocDemo>
  <Checkbox isInvalid>Agree to terms</Checkbox>
</DocDemo>

```tsx
<Checkbox isInvalid>Agree to terms</Checkbox>
```

## Children

The `children` prop can be a string label or a render function that receives the component state, allowing you to conditionally render based on `isSelected` or `isIndeterminate`.

```tsx
<Checkbox>
  {({ isSelected }) => isSelected ? 'Unsubscribe' : 'Subscribe'}
</Checkbox>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Space` | Toggles the checkbox. |
