# Calendar / RangeCalendar

`Calendar` displays date grids for single date selection. `RangeCalendar` allows contiguous date range selection.

## Installation

```bash
rui add calendar
```

## Usage

### Calendar

<DocDemo>
  <Calendar aria-label="Event date" />
</DocDemo>

```tsx
import { Calendar } from '@/components/ui/Calendar';

<Calendar aria-label="Event date" />
```

```tsx

<RangeCalendar aria-label="Trip dates" />
```

## Value

Both components have no selection by default. Provide `defaultValue` for uncontrolled, `value` for controlled.

Date values use [@internationalized/date](https://react-spectrum.adobe.com/internationalized/date/) package.

### Calendar

```tsx
import { Calendar } from '@/components/ui/Calendar';
import { parseDate } from '@internationalized/date';

function ControlledCalendar() {
  const [value, setValue] = useState(parseDate('2024-09-05'));

  return (
    <Calendar aria-label="Event date" value={value} onChange={setValue} />
  );
}
```

### RangeCalendar

<DocDemo>
  <RangeCalendar aria-label="Trip dates" />
</DocDemo>

```tsx
import { parseDate } from '@internationalized/date';
import { RangeCalendar } from '@/components/ui/Calendar';

function ControlledRangeCalendar() {
  const [range, setRange] = useState({
    start: parseDate('2024-02-03'),
    end: parseDate('2024-02-12'),
  });

  return (
    <RangeCalendar
      aria-label="Trip dates"
      value={range}
      onChange={setRange}
    />
  );
}
```

## Label

Use `aria-label` when no visible label present.

```tsx
<Calendar aria-label="Appointment date" />
<RangeCalendar aria-label="Select date range" />
```

## Min and Max Dates

Use `minValue` and `maxValue` to constrain selection.

```tsx
import { today, getLocalTimeZone } from '@internationalized/date';

<Calendar
  aria-label="Appointment date"
  minValue={today(getLocalTimeZone())}
/>

<RangeCalendar
  aria-label="Trip dates"
  minValue={today(getLocalTimeZone())}
/>
```

## Disabled

Use `isDisabled` to disable navigation and selection.

<DocDemo>
  <Calendar aria-label="Event date" isDisabled />
</DocDemo>

```tsx
<Calendar aria-label="Event date" isDisabled />
<RangeCalendar aria-label="Trip dates" isDisabled />
```

## Read Only

Use `isReadOnly` to view selection without allowing changes.

```tsx
<Calendar aria-label="Event date" value={parseDate('2024-09-05')} isReadOnly />

<RangeCalendar
  aria-label="Trip dates"
  value={{ start: parseDate('2024-02-03'), end: parseDate('2024-02-12') }}
  isReadOnly
/>
```

## Unavailable Dates

Use `isDateUnavailable` callback to mark specific dates unavailable. Dates remain focusable but unselectable.

```tsx
import { isWeekend, today, getLocalTimeZone } from '@internationalized/date';

<Calendar
  aria-label="Appointment date"
  isDateUnavailable={(date) => isWeekend(date, 'en-US')}
/>

<RangeCalendar
  aria-label="Trip dates"
  isDateUnavailable={(date) => isWeekend(date, 'en-US')}
/>
```

## Non-Contiguous Ranges (RangeCalendar only)

Use `allowsNonContiguousRanges` to permit ranges containing unavailable dates.

```tsx
<RangeCalendar
  aria-label="Trip dates"
  allowsNonContiguousRanges
  isDateUnavailable={(date) => isWeekend(date, 'en-US')}
/>
```

## Validation (RangeCalendar only)

Use `isInvalid` prop with `errorMessage` slot.

```tsx
<RangeCalendar
  aria-label="Trip dates"
  isInvalid
  errorMessage="Maximum stay is 14 days."
/>
```

## Visible Months (RangeCalendar only)

Display multiple months using `visibleDuration` prop.

```tsx
<RangeCalendar
  aria-label="Trip dates"
  visibleDuration={{ months: 2 }}
/>
```

## International Calendars (Calendar only)

`Calendar` supports 13 calendar systems (Gregorian, Buddhist, Hebrew, Indian, Islamic, etc.). Dates display in locale-appropriate system.

```tsx
import { CalendarDate, createCalendar } from '@internationalized/date';

<Calendar
  aria-label="Appointment date"
  calendar={createCalendar('hebrew')}
/>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Arrow Left` | Move focus to the previous day. |
| `Arrow Right` | Move focus to the next day. |
| `Arrow Up` | Move focus to the same day of the previous week. |
| `Arrow Down` | Move focus to the same day of the next week. |
| `Home` | Move focus to the first day of the month. |
| `End` | Move focus to the last day of the month. |
| `Page Up` | Move focus to the same date in the previous month. |
| `Page Down` | Move focus to the same date in the next month. |
| `Shift + Page Up` | Move focus to the same date in the previous year. |
| `Shift + Page Down` | Move focus to the same date in the next year. |
| `Enter` / `Space` | Select focused date. For `RangeCalendar`, select as start or end of range. |
