# ComboBox

A combobox combines a text input with a dropdown listbox, allowing users to filter a list of options and select a value.

## Installation

```bash
rui add combobox
```

## Usage

<DocDemo>
  <ComboBox label="Favorite animal">
    <ComboBoxItem>Cat</ComboBoxItem>
    <ComboBoxItem>Dog</ComboBoxItem>
    <ComboBoxItem>Horse</ComboBoxItem>
  </ComboBox>

```tsx
import { ComboBox, ComboBoxItem } from '@/components/ui/ComboBox';

<ComboBox label="Favorite animal">
  <ComboBoxItem>Cat</ComboBoxItem>
  <ComboBoxItem>Dog</ComboBoxItem>
  <ComboBoxItem>Horse</ComboBoxItem>
</ComboBox>
```

## Items

Items can be passed as static children for simple lists. The `items` prop with a render function supports dynamic collections.

```tsx
const animals = [
  { id: 1, name: 'Cat' },
  { id: 2, name: 'Dog' },
  { id: 3, name: 'Horse' },
];

<ComboBox label="Animal" items={animals}>
  {(item) => <ComboBoxItem id={item.id}>{item.name}</ComboBoxItem>}
</ComboBox>
```

## Value

Use the `selectedKey` and `onSelectionChange` props to control the selected item. An initial, uncontrolled value can be provided using `defaultSelectedKey`.

```tsx
function ControlledComboBox() {
  const [selected, setSelected] = useState<Key | null>(null);

  return (
    <ComboBox
      label="Animal"
      selectedKey={selected}
      onSelectionChange={setSelected}
    >
      <ComboBoxItem id="cat">Cat</ComboBoxItem>
      <ComboBoxItem id="dog">Dog</ComboBoxItem>
      <ComboBoxItem id="horse">Horse</ComboBoxItem>
    </ComboBox>
  );
}
```

## Validation

Use the `isRequired` prop to require a selection. The `errorMessage` prop provides a custom message displayed when validation fails.

```tsx
<ComboBox
  label="Framework"
  isRequired
  errorMessage="Please select a framework."
>
  <ComboBoxItem>React</ComboBoxItem>
  <ComboBoxItem>Vue</ComboBoxItem>
  <ComboBoxItem>Svelte</ComboBoxItem>
</ComboBox>
```

## Disabled

Use the `isDisabled` prop to disable the entire combobox, including the input and button.

```tsx
<ComboBox label="Size" isDisabled>
  <ComboBoxItem>Small</ComboBoxItem>
  <ComboBoxItem>Medium</ComboBoxItem>
  <ComboBoxItem>Large</ComboBoxItem>
</ComboBox>
```

## Read Only

Use the `isReadOnly` prop to allow viewing the selected value without allowing changes.

```tsx
<ComboBox label="Category" selectedKey="electronics" isReadOnly>
  <ComboBoxItem id="electronics">Electronics</ComboBoxItem>
  <ComboBoxItem id="clothing">Clothing</ComboBoxItem>
</ComboBox>
```

## Custom Filter

By default, the combobox filters items using a "contains" substring match. Use the `defaultFilter` prop or the `inputValue` / `onInputChange` props for custom filtering logic.

```tsx
<ComboBox
  label="Search"
  defaultFilter={(textValue, inputValue) =>
    textValue.toLowerCase().startsWith(inputValue.toLowerCase())
  }
>
  <ComboBoxItem>Apple</ComboBoxItem>
  <ComboBoxItem>Apricot</ComboBoxItem>
  <ComboBoxItem>Banana</ComboBoxItem>
</ComboBox>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Arrow Down` | Opens the listbox and moves focus to the first item or the last selected item. |
| `Arrow Up` | Opens the listbox and moves focus to the last item or the last selected item. |
| `Enter` | Selects the focused item and closes the listbox. |
| `Escape` | Closes the listbox. |
| `Home` | Moves focus to the first item. |
| `End` | Moves focus to the last item. |
