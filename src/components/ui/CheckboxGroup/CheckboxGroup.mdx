# CheckboxGroup

A CheckboxGroup allows users to select one or more items from a list of choices. It manages the selection state and accessibility for a group of related checkboxes.

## Installation

```bash
rui add checkbox-group
```

## Usage

<DocDemo>
  <CheckboxGroup label="Favorite sports">
    <Checkbox value="soccer">Soccer</Checkbox>
    <Checkbox value="baseball">Baseball</Checkbox>
    <Checkbox value="basketball">Basketball</Checkbox>
  </CheckboxGroup>
</DocDemo>

```tsx
import { CheckboxGroup } from '@/components/ui/CheckboxGroup';
import { Checkbox } from '@/components/ui/Checkbox';

<CheckboxGroup label="Favorite sports">
  <Checkbox value="soccer">Soccer</Checkbox>
  <Checkbox value="baseball">Baseball</Checkbox>
  <Checkbox value="basketball">Basketball</Checkbox>
</CheckboxGroup>
```

## Value

Use the `value` and `onChange` props to control the selected items. An initial, uncontrolled value can be provided using `defaultValue`.

```tsx
function ControlledCheckboxGroup() {
  const [selected, setSelected] = useState(['soccer', 'baseball']);

  return (
    <CheckboxGroup
      label="Favorite sports"
      value={selected}
      onChange={setSelected}
    >
      <Checkbox value="soccer">Soccer</Checkbox>
      <Checkbox value="baseball">Baseball</Checkbox>
      <Checkbox value="basketball">Basketball</Checkbox>
    </CheckboxGroup>
  );
}
```

## Forms

Use the `name` prop to submit the selected checkboxes to the server. Set the `isRequired` prop to validate that at least one checkbox is selected.

<DocDemo>
  <CheckboxGroup label="Sandwich condiments" name="condiments" isRequired>
    <Checkbox value="lettuce">Lettuce</Checkbox>
    <Checkbox value="tomato">Tomato</Checkbox>
    <Checkbox value="onion">Onion</Checkbox>
  </CheckboxGroup>
</DocDemo>

```tsx
<CheckboxGroup label="Sandwich condiments" name="condiments" isRequired>
  <Checkbox value="lettuce">Lettuce</Checkbox>
  <Checkbox value="tomato">Tomato</Checkbox>
  <Checkbox value="onion">Onion</Checkbox>
</CheckboxGroup>
```

## Disabled

Use the `isDisabled` prop on the CheckboxGroup to disable all checkboxes within the group. To disable individual checkboxes, use the `isDisabled` prop on each `Checkbox`.

<DocDemo>
  <CheckboxGroup label="Disabled group" isDisabled>
    <Checkbox value="option1">Option 1</Checkbox>
    <Checkbox value="option2">Option 2</Checkbox>
    <Checkbox value="option3">Option 3</Checkbox>
  </CheckboxGroup>
</DocDemo>

```tsx
<CheckboxGroup label="Disabled group" isDisabled>
  <Checkbox value="option1">Option 1</Checkbox>
  <Checkbox value="option2">Option 2</Checkbox>
  <Checkbox value="option3">Option 3</Checkbox>
</CheckboxGroup>
```

## Invalid

Use the `isInvalid` or `errorMessage` prop to indicate a validation error.

<DocDemo>
  <CheckboxGroup

  >
    <Checkbox value="terms">Terms and conditions</Checkbox>
    <Checkbox value="privacy">Privacy policy</Checkbox>
  </CheckboxGroup>
</DocDemo>

```tsx
<CheckboxGroup
  label="Terms and conditions"
  isInvalid
  errorMessage="You must accept at least one option."
>
  <Checkbox value="terms">Terms and conditions</Checkbox>
  <Checkbox value="privacy">Privacy policy</Checkbox>

```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Space` | Toggles the focused checkbox. |
| `Arrow Up` / `Arrow Down` | Moves focus between checkboxes in vertical orientation. |
| `Arrow Left` / `Arrow Right` | Moves focus between checkboxes in horizontal orientation. |
