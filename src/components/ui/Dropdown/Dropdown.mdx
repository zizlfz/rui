# Dropdown

A dropdown menu displays a list of actions or options when triggered by a button. It supports single and multiple selection, keyboard navigation, and nested submenus.

## Installation

```bash
rui add dropdown
```

## Usage

```tsx
import {
  DropdownTrigger,
  Dropdown,
  DropdownItem,
} from '@/components/ui/Dropdown';
import { Button } from '@/components/ui/Button';

<DropdownTrigger>
  <Button>Open Menu</Button>
  <Dropdown>
    <DropdownItem>Cut</DropdownItem>
    <DropdownItem>Copy</DropdownItem>
    <DropdownItem>Paste</DropdownItem>
  </Dropdown>
</DropdownTrigger>
```

## Selection

Use the `selectionMode` prop to enable single or multiple selection. The dropdown manages selection state automatically.

      <DropdownItem id="blue">Blue</DropdownItem>

## Disabled Items

Use the `isDisabled` prop on individual items to disable them.

<DocDemo>
  <DropdownTrigger>
    <Button variant="secondary">Actions</Button>
    <Dropdown>
      <DropdownItem>Save</DropdownItem>
      <DropdownItem>Save As...</DropdownItem>
      <DropdownItem isDisabled>Delete</DropdownItem>
    </Dropdown>
  </DropdownTrigger>
</DocDemo>

```tsx
<DropdownTrigger>
  <Button>Actions</Button>
  <Dropdown>
    <DropdownItem>Save</DropdownItem>
    <DropdownItem>Save As...</DropdownItem>
    <DropdownItem isDisabled>Delete</DropdownItem>
  </Dropdown>
</DropdownTrigger>
```

## Sections

Group related items using `DropdownSection` with a `DropdownTitle` element.

<DocDemo>
  <DropdownTrigger>
    <Button variant="secondary">View Options</Button>
    <Dropdown>

        <DropdownTitle>Appearance</DropdownTitle>
        <DropdownItem>Zoom In</DropdownItem>
        <DropdownItem>Zoom Out</DropdownItem>
        <DropdownItem>Reset Zoom</DropdownItem>
      </DropdownSection>
      <DropdownSection>
        <DropdownTitle>Mode</DropdownTitle>
        <DropdownItem>Light Mode</DropdownItem>
        <DropdownItem>Dark Mode</DropdownItem>
      </DropdownSection>
    </Dropdown>
  </DropdownTrigger>
</DocDemo>

```tsx
<DropdownTrigger>
  <Button>View Options</Button>
  <Dropdown>
    <DropdownSection>
      <DropdownTitle>Appearance</DropdownTitle>
      <DropdownItem>Zoom In</DropdownItem>
      <DropdownItem>Zoom Out</DropdownItem>
      <DropdownItem>Reset Zoom</DropdownItem>
    </DropdownSection>
    <DropdownSection>
      <DropdownTitle>Mode</DropdownTitle>
      <DropdownItem>Light Mode</DropdownItem>
      <DropdownItem>Dark Mode</DropdownItem>
    </DropdownSection>
  </Dropdown>
</DropdownTrigger>
```

Alternatively, use the `title` prop on `DropdownSection` for simple text headers.

```tsx
<DropdownSection title="Appearance">
  <DropdownItem>Zoom In</DropdownItem>
  <DropdownItem>Zoom Out</DropdownItem>
</DropdownSection>
```

## Submenus

Create nested menus using `SubdropdownTrigger`.

<DocDemo>
  <DropdownTrigger>
    <Button variant="secondary">Edit</Button>
    <Dropdown>
      <DropdownItem>Undo</DropdownItem>
      <DropdownItem>Redo</DropdownItem>
      <SubdropdownTrigger>
        <DropdownItem>Find</DropdownItem>
        <Dropdown>
          <DropdownItem>Find...</DropdownItem>
          <DropdownItem>Find Next</DropdownItem>
          <DropdownItem>Find Previous</DropdownItem>
        </Dropdown>
      </SubdropdownTrigger>
    </Dropdown>
  </DropdownTrigger>
</DocDemo>

```tsx
<DropdownTrigger>
  <Button>Edit</Button>
  <Dropdown>
    <DropdownItem>Undo</DropdownItem>

    <SubdropdownTrigger>
      <DropdownItem>Find</DropdownItem>
      <Dropdown>
        <DropdownItem>Find...</DropdownItem>
        <DropdownItem>Find Next</DropdownItem>
        <DropdownItem>Find Previous</DropdownItem>
      </Dropdown>

</DropdownTrigger>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Enter` or `Space` | Opens the menu or activates the focused item. |
| `Arrow Down` | Moves focus to the next item. |
| `Arrow Up` | Moves focus to the previous item. |
| `Arrow Right` | Opens a submenu. |
| `Arrow Left` | Closes a submenu and returns focus to parent. |
| `Escape` | Closes the menu. |
| `Home` | Moves focus to the first item. |
| `End` | Moves focus to the last item. |
