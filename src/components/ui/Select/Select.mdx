# Select

A select displays a collapsible list of options and allows a user to select one of them.

## Installation

```bash
rui add select
```

## Value

Use `value` and `onChange` for controlled selection. The value corresponds to the `id` prop of a `SelectItem`. An initial, uncontrolled value can be provided using `defaultValue`.

```tsx
import { Select, SelectItem } from '@/components/ui/Select';
import { useState } from 'react';

function Example() {
  const [animal, setAnimal] = useState("bison");

  return (
    <Select label="Pick an animal" value={animal} onChange={setAnimal}>
      <SelectItem id="koala">Koala</SelectItem>
      <SelectItem id="kangaroo">Kangaroo</SelectItem>
      <SelectItem id="platypus" isDisabled>Platypus</SelectItem>
      <SelectItem id="eagle">Bald Eagle</SelectItem>
      <SelectItem id="bison">Bison</SelectItem>
      <SelectItem id="skunk">Skunk</SelectItem>
    </Select>
  );
}
```

## Disabled

Use the `isDisabled` prop on the `Select` to disable the entire component, or on individual `SelectItem` elements to disable specific options.

<DocDemo>
  <Select label="Animal" isDisabled>
    <SelectItem id="cat">Cat</SelectItem>
    <SelectItem id="dog">Dog</SelectItem>
  </Select>
</DocDemo>

```tsx
<Select label="Animal" isDisabled>
  <SelectItem id="cat">Cat</SelectItem>
  <SelectItem id="dog">Dog</SelectItem>
</Select>

<Select label="Animal">
  <SelectItem id="cat">Cat</SelectItem>
  <SelectItem id="dog" isDisabled>Dog (unavailable)</SelectItem>
</Select>
```

## Validation

Use `isRequired` to require selection before form submission. Provide `errorMessage` to display a custom error message.

```tsx
<Select label="Animal" isRequired errorMessage="Please select an animal.">
  <SelectItem id="aardvark">Aardvark</SelectItem>
  <SelectItem id="cat">Cat</SelectItem>
  <SelectItem id="dog">Dog</SelectItem>
</Select>
```

## Forms

Use the `name` prop to include the selected item's `id` in form submissions.

```tsx
import { Select, SelectItem } from '@/components/ui/Select';
import { Button } from '@/components/ui/Button';

<form>
  <Select label="Animal" name="animal" isRequired description="Please select an animal.">
    <SelectItem id="aardvark">Aardvark</SelectItem>
    <SelectItem id="cat">Cat</SelectItem>
    <SelectItem id="dog">Dog</SelectItem>
    <SelectItem id="kangaroo">Kangaroo</SelectItem>
    <SelectItem id="panda">Panda</SelectItem>
    <SelectItem id="snake">Snake</SelectItem>
  </Select>
  <Button type="submit">Submit</Button>
</form>
```

## Dynamic Items

Items can be provided via the `items` prop for dynamic rendering with a render function.

```tsx
<Select label="Fruit" items={fruits}>
  {(item) => <SelectItem id={item.id}>{item.name}</SelectItem>}
</Select>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Enter` / `Space` | Opens or closes the select menu. |
| `Arrow Up` / `Arrow Down` | Moves focus to the previous / next option. |
| `Home` / `End` | Moves focus to the first / last option. |
| `Escape` | Closes the select menu. |
