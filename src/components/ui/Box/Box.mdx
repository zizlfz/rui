# Box

A box is a generic container with a subtle elevated background, border, and padding. Use it to group or section content.

## Installation

```bash
rui add box
```

## Usage

<DocDemo>
  <Box>This is a basic box.</Box>
</DocDemo>

```tsx
import { Box } from '@/components/ui/Box';

<Box>Content goes here.</Box>
```

## Custom Classes

Use the `className` prop to apply additional styles. Classes are merged with the base `.box` class via the `cn` utility.

<DocDemo>
  <Box className="max-w-sm">A constrained-width box.</Box>
</DocDemo>

```tsx
<Box className="max-w-sm">A constrained-width box.</Box>
```
