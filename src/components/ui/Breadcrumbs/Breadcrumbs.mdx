# Breadcrumbs

Breadcrumbs display a hierarchy of links to the current page or resource in an application. The last `Breadcrumb` without an `href` is automatically marked as the current page.

## Installation

```bash
rui add breadcrumbs
```

## Usage

<DocDemo>
  <Breadcrumbs>
    <Breadcrumb href="/">Home</Breadcrumb>
    <Breadcrumb href="/products">Products</Breadcrumb>
    <Breadcrumb>Details</Breadcrumb>
  </Breadcrumbs>
</DocDemo>

```tsx
import { Breadcrumbs, Breadcrumb } from '@/components/ui/Breadcrumbs';

<Breadcrumbs>
  <Breadcrumb href="/">Home</Breadcrumb>
  <Breadcrumb href="/products">Products</Breadcrumb>
  <Breadcrumb>Details</Breadcrumb>
</Breadcrumbs>
```

A chevron separator is inserted between each item automatically.

## Dynamic Content

`Breadcrumbs` supports dynamic collections via the `items` prop. Use `onAction` to handle navigation when a breadcrumb is pressed.

```tsx
import { Breadcrumbs, Breadcrumb } from '@/components/ui/Breadcrumbs';
import { type Key } from 'react-aria-components';

const items = [
  { id: 1, label: 'Home' },
  { id: 2, label: 'Products' },
  { id: 3, label: 'Details' },
];

<Breadcrumbs items={items} onAction={(key: Key) => navigate(key)}>
  {(item) => <Breadcrumb>{item.label}</Breadcrumb>}
</Breadcrumbs>
```

## Accessibility

When breadcrumbs are used as a main navigation element, place them inside a `<nav>` with an `aria-label` to create a [navigation landmark](https://www.w3.org/WAI/ARIA/apg/patterns/landmarks/examples/navigation.html).

```tsx
<nav aria-label="Breadcrumb">
  <Breadcrumbs>
    <Breadcrumb href="/">Home</Breadcrumb>
    <Breadcrumb href="/products">Products</Breadcrumb>
    <Breadcrumb>Details</Breadcrumb>
  </Breadcrumbs>
</nav>
```
