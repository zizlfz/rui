# Switch

A toggle switch that allows users to turn an option on or off. It provides a visual indication of the current state.

## Installation

```bash
rui add switch
```

## Usage

<DocDemo>
  <Switch>Enable notifications</Switch>
</DocDemo>

```tsx
import { Switch } from '@/components/ui/Switch';

<Switch>Enable notifications</Switch>
```

## Value

Use the `isSelected` and `onChange` props to control the state.

<DocDemo props={{ isSelected: false }} propsForm={{ isSelected: Boolean }}>
  {(props) => <Switch {...props}>Dark mode</Switch>}
</DocDemo>

```tsx
function ControlledSwitch() {
  const [isSelected, setIsSelected] = useState(false);

  return (
    <Switch isSelected={isSelected} onChange={setIsSelected}>
      Dark mode
    </Switch>
  );
}
```

An initial, uncontrolled value can be provided using the `defaultSelected` prop.

```tsx
<Switch defaultSelected>Dark mode</Switch>
```

## Disabled

Use the `isDisabled` prop to disable the switch. It cannot be interacted with and is excluded from the tab order.

<DocDemo>
  <Switch isDisabled>Disabled option</Switch>
</DocDemo>

```tsx
<Switch isDisabled>Disabled option</Switch>
```

## Read Only

Use the `isReadOnly` prop to allow viewing the state without allowing changes.

```tsx
<Switch isSelected isReadOnly>Read-only option</Switch>
```

## Children

The `children` prop can be a string label or a render function that receives the component state, allowing you to conditionally render based on `isSelected`.

```tsx
<Switch>
  {({ isSelected }) => isSelected ? 'On' : 'Off'}
</Switch>
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Space` | Toggles the switch. |
