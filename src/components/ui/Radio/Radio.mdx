# Radio

A RadioGroup with Radio buttons allows users to select a single item from a list of choices. Only one radio button in a group can be selected at a time.

## Installation

```bash
rui add radio
```

## Usage

<DocDemo>
  <RadioGroup label="Favorite color">
    <Radio value="red">Red</Radio>
    <Radio value="green">Green</Radio>
    <Radio value="blue">Blue</Radio>
  </RadioGroup>
</DocDemo>

```tsx
import { RadioGroup, Radio } from '@/components/ui/Radio';

<RadioGroup label="Favorite color">
  <Radio value="red">Red</Radio>
  <Radio value="green">Green</Radio>
  <Radio value="blue">Blue</Radio>
</RadioGroup>
```

## Value

Use the `value` and `onChange` props to control the selected item. An initial, uncontrolled value can be provided using `defaultValue`.

```tsx
function ControlledRadioGroup() {
  const [selected, setSelected] = useState('green');

  return (
    <RadioGroup
      label="Favorite color"
      value={selected}
      onChange={setSelected}
    >
      <Radio value="red">Red</Radio>
      <Radio value="green">Green</Radio>
      <Radio value="blue">Blue</Radio>
    </RadioGroup>
  );
}
```

## Orientation

Use the `orientation` prop to arrange radios horizontally or vertically. The default is `vertical`.

<DocDemo>
  <RadioGroup label="Favorite color" orientation="horizontal">

    <Radio value="green">Green</Radio>
    <Radio value="blue">Blue</Radio>
  </RadioGroup>
</DocDemo>

```tsx
<RadioGroup label="Favorite color" orientation="horizontal">
  <Radio value="red">Red</Radio>
  <Radio value="green">Green</Radio>
  <Radio value="blue">Blue</Radio>
```
```tsx
</RadioGroup>
```

## Forms

Use the `name` prop to submit the selected value to the server. Set the `isRequired` prop to validate that an option is selected.

<DocDemo>
  <RadioGroup label="Shipping method" name="shipping" isRequired>
    <Radio value="standard">Standard (5-7 days)</Radio>
    <Radio value="express">Express (2-3 days)</Radio>
    <Radio value="overnight">Overnight (1 day)</Radio>
  </RadioGroup>
</DocDemo>

```tsx
<RadioGroup label="Shipping method" name="shipping" isRequired>
  <Radio value="standard">Standard (5-7 days)</Radio>
  <Radio value="express">Express (2-3 days)</Radio>
  <Radio value="overnight">Overnight (1 day)</Radio>
</RadioGroup>
```

## Disabled

Use the `isDisabled` prop on the RadioGroup to disable all radios within the group. To disable individual radios, use the `isDisabled` prop on each `Radio`.

<DocDemo>
  <RadioGroup label="Disabled group" isDisabled>
    <Radio value="option1">Option 1</Radio>
    <Radio value="option2">Option 2</Radio>
    <Radio value="option3">Option 3</Radio>
  </RadioGroup>
</DocDemo>

```tsx
<RadioGroup label="Disabled group" isDisabled>
  <Radio value="option1">Option 1</Radio>
  <Radio value="option2">Option 2</Radio>
  <Radio value="option3">Option 3</Radio>
</RadioGroup>
```

Use the `isInvalid` or `errorMessage` prop to indicate a validation error.

    <Radio value="credit">Credit Card</Radio>
    <Radio value="debit">Debit Card</Radio>
    <Radio value="paypal">PayPal</Radio>

```tsx
<RadioGroup

  <Radio value="credit">Credit Card</Radio>
  <Radio value="debit">Debit Card</Radio>

```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Space` | Selects the focused radio. |
| `Arrow Up` / `Arrow Down` | Moves focus between radios in vertical orientation. |
| `Arrow Left` / `Arrow Right` | Moves focus between radios in horizontal orientation. |
