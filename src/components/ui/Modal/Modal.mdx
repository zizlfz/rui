# Modal

A modal is an overlay element that blocks interaction with content outside it. It renders a backdrop with a centered dialog box.

## Installation

```bash
rui add modal
```

## Usage

<DocDemo>
  <DialogTrigger>
    <Button>Open Modal</Button>
    <Modal>
      <Dialog>
        <Heading slot="title">Modal Title</Heading>
        <p>Modal content goes here.</p>
        <Button slot="close" variant="secondary">Close</Button>
      </Dialog>
    </Modal>
  </DialogTrigger>
</DocDemo>

```tsx
import { Modal, Dialog } from '@/components/ui/Modal';
import { Button } from '@/components/ui/Button';

<DialogTrigger>
  <Button>Open Modal</Button>
  <Modal>
    <Dialog>
      <Heading slot="title">Modal Title</Heading>
      <p>Modal content goes here.</p>
      <Button slot="close">Close</Button>
    </Dialog>
  </Modal>
</DialogTrigger>
```

Modal uses `DialogTrigger` from react-aria-components to manage open/close state. Wrap the trigger and modal in a `DialogTrigger`, and use `slot="close"` on any button inside to close the modal.

## Dismissable

Use the `isDismissable` prop to allow closing the modal by clicking outside or pressing Escape.

<DocDemo props={{ isDismissable: true }} propsForm={{ isDismissable: Boolean }}>
  {(props) => (
    <DialogTrigger>
      <Button>Open Dismissible</Button>
      <Modal {...props}>
        <Dialog>
          <Heading slot="title">Dismissible Modal</Heading>
          <p>Click outside or press Escape to close.</p>
        </Dialog>
      </Modal>
    </DialogTrigger>
  )}
</DocDemo>

```tsx
<DialogTrigger>
  <Button>Open Modal</Button>
  <Modal isDismissable>
    <Dialog>
      <Heading slot="title">Dismissible Modal</Heading>
      <p>Click outside or press Escape to close.</p>
    </Dialog>
  </Modal>
</DialogTrigger>
```

## Controlled

Use the `isOpen` and `onOpenChange` props to control the modal programmatically.

```tsx
import { useState } from 'react';
import { Modal, Dialog } from '@/components/ui/Modal';
import { Button } from '@/components/ui/Button';

function Example() {
  const [isOpen, setOpen] = useState(false);

  return (
    <>
      <Button onPress={() => setOpen(true)}>Open Modal</Button>
      <Modal isOpen={isOpen} onOpenChange={setOpen}>
        <Dialog>
          <Heading slot="title">Controlled Modal</Heading>
          <p>This modal is controlled by React state.</p>
          <Button slot="close" onPress={() => setOpen(false)}>Close</Button>
        </Dialog>
      </Modal>
    </>
  );
}
```

## Keyboard Interactions

| Key | Description |
| --- | --- |
| `Escape` | Closes the modal (unless `isKeyboardDismissDisabled` is set). |
| `Tab` | Moves focus to the next focusable element inside the modal. |
| `Shift + Tab` | Moves focus to the previous focusable element inside the modal. |

## Accessibility

The modal renders with `role="dialog"` and `aria-modal="true"`. When open, focus is trapped inside the modal and returns to the trigger element when closed.
